## launcher
A launcher app that allows users to choose favorites from a list of launchable apps.


## Libraries/frameworks used
[Android Jetpack(Room, Databinding, ViewModels, and Navigation Component)](https://developer.android.com/jetpack)  
[Dagger](https://github.com/google/dagger)  
[RxJava2](https://github.com/ReactiveX/RxJava)  
[Drag & Drop n' Swipe Recyclerview](https://github.com/ernestoyaquello/DragDropSwipeRecyclerview)  
[Moshi (for serialization of objects)](https://github.com/square/moshi)
